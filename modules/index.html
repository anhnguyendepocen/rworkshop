Directory structure

All modules assume the following directory structure:

rworkshop/
|
|__ data/
|   |
|   |-- module1.data
|   |-- module2.data
|   |...
|
|__ scripts/
|   |
|   |-- module1.R
|   |-- module2.R
|   |...

For all scripts, the working directory should be path/to/scripts.
